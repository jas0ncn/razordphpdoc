# 错误代码

本文介绍 Razord 的所有错误代码。

## `1xx` 系统错误

### `101`

PHP 版本过低，Razord PHP 支持最低的 PHP 版本为 `5.3.0`，如果你的服务器的 PHP 版本低于 `5.3.0` ，将会报告 `100` 错误。

### `102`

调用 `load()` 函数时参数不能为空。详细请查看 [API文档](apis.md) 。

### `103`

载入的模块未找到，请将模块放在 `./module` 文件夹内。详细请查看 [模块系统](module.md) 。

### `104`

载入的模块名称使用了保留字。关于保留字请查看 [模块系统](module.md) 。
